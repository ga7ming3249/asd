# Label Rules

## Product

| Label | Description |
|---|---|
| `product:reference-assistant` | Reference Assistant plugin |
| `product:type-inventory` | Type Inventory plugin |
| `product:color-inventory` | Color Inventory plugin |
| `product:component-package` | Component Package plugin |

## Priority

| Label | Description |
|---|---|
| `priority:P0` | Critical — blocks release or causes data loss |
| `priority:P1` | High — significant impact, should fix soon |
| `priority:P2` | Medium — noticeable issue, fix in next cycle |
| `priority:P3` | Low — nice to have, backlog |

## Status

| Label | Description |
|---|---|
| `status:open` | Needs triage or not yet started |
| `status:in-progress` | Actively being worked on |
| `status:review` | Implementation done, under review |
| `status:done` | Completed and verified |
